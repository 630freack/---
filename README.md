# Монитор интернет-трафика

Приложение на Python с графическим интерфейсом, которое отслеживает и отображает интернет-трафик и текущую скорость передачи данных для каждого непосистемного приложения в реальном времени.

## Возможности

- Отображение списка всех активных приложений, использующих интернет
- Показ текущей скорости загрузки и отдачи для каждого приложения
- Отображение общего объёма переданных данных
- Фильтрация системных процессов
- Обновление данных каждые 1-2 секунды
- Визуальное выделение приложений с высокой скоростью трафика (красным цветом)
- Кнопки управления: "Запустить", "Остановить", "Обновить"
- Корректная обработка ошибок и завершения процессов

## Требования

- Python 3.7 или выше
- Библиотека psutil
- Графическая библиотека tkinter (входит в стандартную библиотеку Python)

## Установка

1. Установите Python с официального сайта: https://www.python.org/downloads/

2. Установите необходимые зависимости:
```bash
pip install -r requirements.txt
```

## Запуск приложения

1. Запустите приложение из командной строки:
```bash
python src/main.py
```

2. Для корректной работы рекомендуется запускать приложение от имени администратора, так как некоторые процессы могут требовать повышенных привилегий для доступа к сетевой статистике.

## Использование

1. Нажмите кнопку "Запустить" для начала мониторинга
2. Данные будут обновляться автоматически каждую секунду
3. Кнопка "Обновить" позволяет обновить данные вручную
4. Нажмите "Остановить" для приостановки мониторинга
5. Приложения с высокой скоростью трафика (>1 МБ/с) будут выделены красным цветом

## Конфигурация

Вы можете изменить следующие параметры в файле `src/main.py`:

- `UPDATE_INTERVAL` - интервал обновления данных (в секундах)
- `HIGH_TRAFFIC_THRESHOLD` - порог высокого трафика в байтах/сек (по умолчанию 1 МБ/с)
- `SYSTEM_PROCESSES` - список системных процессов, которые следует фильтровать

## Примечания

- Приложение может не отображать все процессы без прав администратора
- Некоторые системные процессы могут временно использовать интернет, но они автоматически фильтруются
- Общий объём данных считается с момента запуска приложения
- Приложение использует psutil для получения информации о сетевой активности процессов
